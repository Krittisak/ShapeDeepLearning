# ShapeDeepLearning
Deep learning of shapes

## Command-Line

### Plot a Circle:
    python main.py -d 2 -s sphere

### Plot a Sphere:
    python main.py -d 3 -s sphere

### Plot a Torus:
    python main.py -d 3 -s torus
